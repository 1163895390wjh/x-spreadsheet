## Install
``` shell
npm install x-spreadsheet
``` 

## Quick Start
``` html
<div id="example"></div>
```

``` javascript
import Spreadsheet from 'x-spreadsheet';
new Spreadsheet(document.getElementById('example')).loadData({});
```
